## 🚀 About Me

Full-stack developer with **4+ years** of experience building digital products that are not just functional, but genuinely make an impact — from platforms serving millions of users to smooth, secure web applications. Driven by one idea: technology should be **powerful and inclusive**.

- 🔭 Currently building the digital platform at **TaxAssist**
- ♿ Accessibility advocate — shipped **AA / AAA** compliant sites
- 🧩 I care about clean, maintainable, type-safe code
- 🌍 Worked on projects for **HS2, UK Parliament, Kinship, The Fitzwilliam Museum**
